1. Instalar o node v16.10.0

2. Instalar o Git

3. Clonar o repositorio em uma pasta bem sossegada (tipo C:/printerServer)
git clone https://github.com/RafaelBini/RichotRestaurantePrinterServer.git

4. Rodar o npm ci

5. Colocar o arquivo .env

6 Instalar pm2 (npm install -g pm2)

7 Instalar pm2-windows-service (npm install -g pm2-windows-service)

8 pm2 start richotPrinterServer.js

9 pm2 save
